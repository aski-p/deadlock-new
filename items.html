아이템 API 엔드포인트

GET /api/items
모든 아이템 목록을 조회합니다.
GET /api/items/{id}
특정 아이템의 상세 정보를 조회합니다.
GET /api/items/categories
아이템 카테고리 목록을 조회합니다.
GET /api/items/builds
추천 아이템 빌드를 조회합니다.
// JavaScript 예제
fetch('https://deadlock-api.com/api/items')
  .then(response => response.json())
  .then(items => {
    console.log('아이템 목록:', items);
    items.forEach(item => {
      console.log(`${item.name} - ${item.cost}원`);
    });
  });

// 카테고리별 아이템 조회
fetch('https://deadlock-api.com/api/items/categories/weapon')
  .then(response => response.json())
  .then(weapons => {
    console.log('무기 아이템:', weapons);
  });

Deadlock API

게임 데이터, 이미지, 통계 등을 제공하는 포괄적인 API

시작하기 API 탐색

주요 기능

Deadlock API는 게임 개발자와 데이터 분석가를 위한 다양한 기능을 제공합니다.

영웅 데이터

모든 영웅의 상세 정보, 능력, 통계 및 이미지를 제공합니다.

GET /heroes

아이템 데이터

게임 내 모든 아이템의 정보, 효과, 가격 등을 제공합니다.

GET /items

매치 데이터

게임 매치 결과, 플레이어 통계, 순위 정보를 제공합니다.

GET /matches

실시간 데이터

실시간 게임 상태, 서버 정보, 플레이어 활동을 제공합니다.

GET /live

이미지 API

영웅, 아이템, 능력 아이콘 등 고화질 이미지를 제공합니다.

GET /images

통계 분석

게임 메타, 승률, 픽률 등 다양한 통계 데이터를 제공합니다.

GET /stats

API 문서

Deadlock API를 사용하여 강력한 애플리케이션을 만들어보세요.

빠른 시작 가이드

1. API 키 받기

Deadlock API를 사용하려면 먼저 API 키를 받아야 합니다. 여기서 가입하여 무료 API 키를 받으세요.

2. 첫 번째 API 호출

모든 영웅 목록을 가져오는 간단한 예제입니다:

curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://deadlock-api.com/api/heroes
3. 응답 형식

모든 API 응답은 JSON 형식으로 제공됩니다:

{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "인페르누스",
      "role": "캐리",
      "abilities": [...],
      "stats": {...}
    }
  ],
  "meta": {
    "total": 25,
    "page": 1,
    "per_page": 10
  }
}
4. 오류 처리

API 오류는 HTTP 상태 코드와 함께 상세한 정보를 제공합니다:

{
  "success": false,
  "error": {
    "code": 401,
    "message": "인증이 필요합니다",
    "details": "유효한 API 키를 제공해주세요"
  }
}

사용 예제

다양한 프로그래밍 언어로 작성된 API 사용 예제를 확인하세요.

JavaScript (Node.js)

const axios = require('axios');

// API 키 설정
const API_KEY = 'your-api-key-here';
const BASE_URL = 'https://deadlock-api.com/api';

// 모든 영웅 조회
async function getHeroes() {
  try {
    const response = await axios.get(`${BASE_URL}/heroes`, {
      headers: {
        'Authorization': `Bearer ${API_KEY}`
      }
    });
    
    console.log('영웅 목록:', response.data.data);
    return response.data.data;
  } catch (error) {
    console.error('API 호출 오류:', error.response?.data || error.message);
  }
}

// 특정 영웅 상세 정보
async function getHeroDetails(heroId) {
  try {
    const response = await axios.get(`${BASE_URL}/heroes/${heroId}`, {
      headers: {
        'Authorization': `Bearer ${API_KEY}`
      }
    });
    
    console.log('영웅 상세 정보:', response.data.data);
    return response.data.data;
  } catch (error) {
    console.error('API 호출 오류:', error.response?.data || error.message);
  }
}

// 사용 예제
(async () => {
  await getHeroes();
  await getHeroDetails(1);
})();

Python

import requests
import json

# API 키 설정
API_KEY = 'your-api-key-here'
BASE_URL = 'https://deadlock-api.com/api'

def get_heroes():
    """모든 영웅 조회"""
    headers = {
        'Authorization': f'Bearer {API_KEY}'
    }
    
    try:
        response = requests.get(f'{BASE_URL}/heroes', headers=headers)
        response.raise_for_status()
        
        data = response.json()
        print('영웅 목록:', json.dumps(data['data'], indent=2, ensure_ascii=False))
        return data['data']
    
    except requests.exceptions.RequestException as e:
        print(f'API 호출 오류: {e}')
        return None

def get_hero_stats(hero_id):
    """영웅 통계 조회"""
    headers = {
        'Authorization': f'Bearer {API_KEY}'
    }
    
    try:
        response = requests.get(f'{BASE_URL}/heroes/{hero_id}/stats', headers=headers)
        response.raise_for_status()
        
        data = response.json()
        print(f'영웅 {hero_id} 통계:', json.dumps(data['data'], indent=2, ensure_ascii=False))
        return data['data']
    
    except requests.exceptions.RequestException as e:
        print(f'API 호출 오류: {e}')
        return None

# 사용 예제
if __name__ == '__main__':
    heroes = get_heroes()
    if heroes and len(heroes) > 0:
        hero_stats = get_hero_stats(heroes[0]['id'])

🚀 API 사용 예제

간단한 API 호출로 Deadlock 데이터를 가져올 수 있습니다:

// 모든 영웅 정보 가져오기
fetch('https://deadlock-api.com/api/heroes')
  .then(response => response.json())
  .then(data => {
    console.log('영웅 데이터:', data);
  });

// 특정 영웅 정보 가져오기
fetch('https://deadlock-api.com/api/heroes/1')
  .then(response => response.json())
  .then(hero => {
    console.log('영웅 정보:', hero.name);
  });
